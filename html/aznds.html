Azion Design System

Design System

Concepção de uma colecção de recursos pré-construídos e reutilizáveis - fundações, componentes, padrões, orientação, e código - que permite criar experiências digitais consistentes.

Sobre

Um Design System unificado é essencial para construir produtos digitais melhores e mais rápidos; melhor porque uma experiência coesa é mais facilmente compreendida pelos nossos utilizadores, e mais rápida porque nos proporciona uma linguagem comum para trabalharmos.

Função

Product Designer

Entregáveis

Design Tokens, Layout Foundation, Components, Handoff to Developers, UI Design, Prototypes.
